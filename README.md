# Coding_Blog_Project

**Coding_Blog_Project** is a full-stack web application built with **Next.js** and **TailwindCSS**. It showcases dynamic blog posts, responsive UI, dark/light mode, contact forms, FAQ section, and skill highlights. Ideal for learning modern web development practices.

## Features

- Dynamic blog posts with images and descriptions  
- Responsive and mobile-friendly design  
- Dark and light theme toggle  
- Contact form with validation  
- FAQ section using Accordion  
- Skills showcase with animated cards  

## Technologies Used

- **Next.js** – React framework for server-side rendering  
- **TailwindCSS** – Utility-first CSS framework  
- **Framer Motion** – Smooth animations  
- **React Hook Form** – Form handling and validation  
- **Sonner** – Notifications
